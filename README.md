# GU Boilerplate
## Recommended Env
- node: `10.16.3`
- npm: `^6.9.0`

## Installation
```
$ brew install webp
$ npm i
```

## Usage
### Building bundle with webpack (development mode).
```
$ npm run dev
```

### Building bundle with webpack (production mode).
```
$ npm run build
```

### Building bundle with webpack (preview server mode).
```
$ npm run preview
```
